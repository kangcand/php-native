folder admin adalah tempat pengolahan data
file login untuk user masuk ke dalam pengolahan data atau halaman admin
file logout untuk menghapus sesi login
file index untuk membuat halaman awal 
folder assets untuk memasukan css, js, img, dll (tempat template project)
folder models untuk menyimpan konfigurasi db dan juga logic dari tiap table
users.sql adalah contoh table user yang akan digunakan untuk login ke dalam sistem
